# PDF 扫描件处理工作台

一个面向扫描 PDF 的桌面工具，按固定流程完成：
合并 -> 页码 -> 目录/书签 -> 压缩。

## 功能概览

1. **步骤 1：合并 PDF**
   - 支持一次选择多个 PDF。
   - 支持拖拽调整顺序。
   - 右侧显示选中文件第一页预览。

2. **步骤 2：添加页码**
   - 页码位置：左下角 / 下部中间 / 右下角。
   - 页码参数：字体、字号、颜色。
   - 对已经添加过页码的 PDF 重新编号时，会先用贴合数字外接框的白色遮罩覆盖旧页码，再写入新页码，避免页码重叠。
   - 可点击“上一步”撤销当前页码处理结果，返回步骤 1 重新合并。

3. **步骤 3：编写目录并生成书签**
   - 自动生成初始目录（按合并文件名 + 起始页）。
   - 可手动增删目录项和页码。
   - 选中目录项后，可点击“↑”或“↓”调整目录项顺序。
   - 可设置目录标题字体/字号、目录正文字体/字号。
   - 目录页会插入到正文前面，目录本身不加页码。
   - 生成 PDF 时自动写入书签，点击书签可跳转对应页。
   - 可点击“上一步”撤销已添加页码状态，返回步骤 2 重新设置页码。

4. **步骤 4：压缩输出**
   - 输入目标大小（MB）。
   - 可设置允许超出目标比例（%），减少过度压缩。
   - 压缩后保留目录书签跳转能力。
   - 可点击“上一步”撤销目录生成状态，返回步骤 3 重新编辑目录。

## 使用方法（推荐）

1. 打开程序后，先在 **步骤 1** 选择并排序 PDF，点击“合并并进入步骤 2”。
2. 在 **步骤 2** 设定页码样式，点击“添加页码并进入步骤 3”。
3. 在 **步骤 3** 检查自动目录，按需细调目录项和页码，设置输出路径后点击“生成未压缩完整版并进入压缩”。
4. 在 **步骤 4** 设置目标大小和允许超出比例，点击“压缩并输出最终文件”。

步骤 2、步骤 3、步骤 4 均提供“上一步”按钮，用于撤销当前阶段的处理状态并回到上一流程重新操作。已导出到用户指定路径的 PDF 不会被程序自动删除。

## 输出文件

程序最终只输出两份完整文件：

- 未压缩完整版 PDF（含目录与书签）
- 压缩版完整版 PDF（含目录与书签）

中间过程文件在临时目录中处理，关闭程序后自动清理。

## 运行方式

### 方式 A：直接使用 EXE（Windows 用户）

运行 `dist\PDF_Page_Number_Tool.exe`。

### 方式 B：开发环境运行

```bash
python pdf.py
```

## 打包 EXE

双击运行 `build_exe.bat`。

打包特点：

- 使用项目图标 `app_icon.ico`。
- 排除无关大模块，尽量减小体积。
- 若默认 exe 被占用，自动回退命名。

默认输出：

`dist\PDF_Page_Number_Tool.exe`

回退输出（默认文件被占用时）：

`dist\PDF_Page_Number_Tool_logo.exe`
